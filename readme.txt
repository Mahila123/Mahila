Fitness and Health Calculator
This is a simple fitness and health calculator application that allows users to calculate metrics such as BMI, body fat, ideal weight, 
and calories burned. It consists of a Node.js server and an HTML web page.

Setup and Execution

=> Install Node.js if not already installed: https://nodejs.org
=> Clone or download the project repository.
=> Navigate to the project directory in the terminal.
=> Install dependencies: npm install express
=> Start the server: node server.js
=> Open http://localhost:3000 in your web browser.
=> Fill in the input fields and click the corresponding button to calculate.
=> The results will be displayed on the web page.

Files
server.js: Node.js server code.
index.html: Web page for user interaction.
style.css: CSS styles for the web page.
script.js: JavaScript code for AJAX requests and result display.
